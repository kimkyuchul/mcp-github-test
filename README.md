# MCP GitHub 테스트 프로젝트

## 프로젝트 소개

이 저장소는 MCP(Model Context Protocol)와 연동된 Rust 기반 프로젝트입니다. 

### 주요 기능
- Rust 기본 프로젝트 구조
- MCP 연동 테스트

### 시작하기

1. Rust 설치
   ```bash
   curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
   ```

2. 프로젝트 클론
   ```bash
   git clone https://github.com/kimkyuchul/mcp-github-test.git
   cd mcp-github-test
   ```

3. 프로젝트 실행
   ```bash
   cargo run
   ```
